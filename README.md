## Getting Started

First, Setup the project

- Create a .env file at the root level of the project (Same level as package.json) and it should inclued a single variable called SESSION_SECRET=. The value that should be placed next to the equals can be generated by running the command "openssl rand -base64 32" and put it in quotation marks. An example looks like SESSION_SECRET="BwvcHMoegmn1OMI5tdoVwG6Ud0m9lEYzgEOHJHn+wjc="

-  Next run npm i to install all packages you should only need to do this once but if you even encounter an error where a package is not found try running npm i again.

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

The default login for VosynVerse is Username: a@a.com, Password: a@a12345

If you ever launch the project and get an error 404 and the url is just localhost:3000 try adding /auth to the url to force it to navigate to the login page or try clearing your session tokens.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
